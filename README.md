# python-basic-learning



## Introdusaun 
**Pyhton**
python mak high lever programming language ida ne'ebe dezena atu nune'e fasil ba ema hodi lee no komprendde. liu husi nia sintax sira ne'ebe simples to fasil ba sira ne'ebe foin primerira vez aprende programa.

benefisiu prinsipal pyhton mak ninia library ne'ebe boot tebes no mos komunitas ne'ebe ativu, tan buat sira ne'e mak halo ema barak mak uza pyhton hanesan: estudante, analizador sira no developer sira, to companha teknologia ne'ebe boot.

iha repo ida ne'e python utiliza hodi fasilita atu aprende programa basica sira hanesan variable, tipe data, kondisaun, looping no mos konseitu programa sira seluk ho step ba step.
