## Introdusaun 
**Pyhton**
> mak programming language level ass ida ne'ebe dezenado atu nune'e fasil ba ema hodi lee no komprendde. liu husi nia sintax(kode sira ne'ebe ita hakerek iha file laran) sira ne'ebe simples no fasil ba sira ne'ebe foin primerira vez aprende programa.

> benefisiu prinsipal pyhton mak ninia library ne'ebe barak tebes no mos komunitas ne'ebe ativu, tan buat sira ne'e mak halo ema barak mak uza pyhton hanesan: estudante, analizador sira no developer sira, to companha teknologia ne'ebe boot mos uza python hodi halo sira nia projetu.

> iha repo ida ne'e python utiliza hodi fasilita atu aprende programa basica sira hanesan variable, tipe data, kondisaun, looping no mos konseitu programa sira seluk ho step ba step.

> **karik ita iha interese atu aprende keta haluha loke kada file karik make sure hakerek kode idak idak atu nune'e bele aprende tuir dalan**
